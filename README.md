# Zia Wine Launcher

Simple wine launcher application, for use with wine that has been installed in the system.

## Requirements

- Gnome Terminal
- Wine 32bit
- Python 3
- Tkinter Library

## Running

### Linux

```shell
python3 zia_wine_launcher.pyw
```
